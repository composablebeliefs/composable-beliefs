# Deprecated plans

Plans built on Composable Beliefs schema features that were later removed.
Kept for design history; **do not treat as current**.

- `composable-beliefs-thesis-pt2.md` — earned confidence + the `patch` assertion
  kind + a Bittensor evaluation network. Both the `patch` kind and the
  `confidence` field were removed from the schema, so the central mechanism no
  longer exists in the framework. The decentralized-evaluation ideas are
  preserved here but are not wired to the implementation.
